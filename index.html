Git 命令复制工具

1. 初始化 git 储存
git init
已复制!
2. 需要提交的所有修改放到暂存区
git add .
已复制!
3. 将缓存区的文件，提交到本地仓库（版本库）
git commit -m ""
已复制!
4. 查看工作区、暂存区的状态
git status
已复制!
5. 查看历史提交(commit)记录
git log
已复制!
6. 版本回退（~ 后面的数字表示回退多少个版本）
git reset --hard HEAD~1
已复制!
7. 回到指定哈希值对应的版本
git reset --hard <Hash>
已复制!
8. 强制推送到服务器
git push --force origin master
已复制!
